Tämä projekti oli ohjelmoinnin perusteet -kurssin loppuprojekti

Ohjelmalla on tarkoitus hallinnoida autokorjaamon varaosia. Ohjelmassa on graafinen käyttöliittymä jossa voi lisätä, selata, muokata ja poistaa varaosia. Lisäksi ohjelma voi generoida varaosalistasta CSV-tiedoston. 

Ohjelmassa on kaksi eri kooditiedostoa:
gui_varaosaohjelma.py
varaosaohjelma.py

Ohjelma käynnistetään ajamalla tiedosto gui_varaosaohjelma.py joka tekee kutsuja toiselle tiedostolle.

Ohjelma tallentaa annetut varaosat tiedostoon: varaosa_data.db (luodaan jos ei olemassa)
